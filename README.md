# Art Gallery

A simple Android MVVM application that implements a listing and detailing of artworks from the Art Institute of Chicago's public API.

API: https://api.artic.edu/docs/#quick-start

Main technologies:

- Jetpack
    - Compose
    - Navigation
- Coil
- Ktor
- Koin
- Junit
- Mockk
